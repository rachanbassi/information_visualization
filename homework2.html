Homework 2: Multivariate Data Visualization

Functionality

Basic:

  • Implemented a scatter plot of life expectancy vs income.
  • Changed the size to indicate population of the state.
  • Changed opacity of the bubbles.
  • Included a mouse hover that displays the name of state and its population when hovered over a particular bubble.

Moderate:

  • Created a scatter plot matrix of income, illiteracy, life expectancy, and percentage of high school graduates.
  • Encoded states with colour.
  • Used brushing functionality that allows selecting a region on a particular scatter plot and highlights the corresponding points on all the other graphs.

Advanced:

  • Created a parallel coordinates visualization of population, income, illiteracy, life expectancy, murder, percentage of high school graduates, frost, total area and region.
  • Encoded the region by colour of the path.
  • Used brushing which allows the user to select a portion on an axis and highlights the lines in the selected portion.

Discussion

The data used in the above visualizations comes from the state.x77 data in R. The first graph is a bubble chart of the life expectancy versus income for a particular state. The size of each bubble represents the population of the state and its colour indicates the region of the state. I chose a low opacity for the bubbles to avoid losing smaller bubbles due to overlap. Lower opacity enables the user to get a clearer and complete understanding of the data. I also added the functionality that allows the state and population value to be displayed when the mouse is hovered over the bubble. This allows for more data density - I have state, population, region, life expectancy and income all packed in a single bubble chart.

The second graph is a scatterplot matrix of income, illiteracy, life expectancy, and the percentage of high school graduates. The colour of the circles represents the region to which a particular state belongs. I added the brushing functionality as it provides a much nicer user experience by allowing the user to select the region of the plot or particular data points they are interested and how they compare across the other graphs in the scatter plot matrix.

The third graph is a parallel coordinates implementation of the data. I have mapped every row in the data set for population, income, illiteracy, life expectancy, murder, percentage of high school graduations, frost (number of days with minimum temperature between freezing), total area of a state and the region to which a state belongs. I chose to implement a brushing feature which allows the users to select a particular part of a variable on one of the parallel coordinates and highlights the line connecting the data points which fall in the selected region. Further subsetting can be done by selecting different portions on other variables of the parallel coordinates. The colour of every line indicates the region to which the state belongs. I have chosen a consistent colour scheme as in the bubble chart and scatterplot matrix to make comparison easier.
