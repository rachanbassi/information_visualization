Functionality

Basic:

  • Implemented a line and multi-series line plot of the number of number of drivers, front and rear seat passengers killed.
  • Impemented an area plot of drivers killed.

Moderate:

  • Implemented panning on the area plot which allows the user to click and span rectangular region that they wish to focus one.
  • Created a Details-on-Demand feature on the single time series line plot of the drivers killed. This feature displays a tool tip showing the month and the average value of deaths for that particular month.

Advanced:

  • Implemented brushing on the single area plot of the number of drivers killed.

Discussion

The data used in the above visualizations comes from the Seatbelts dataset in R. Initial prepocessing of the data was conducted by creating a time series object in R and then formatting the date in d3. The first graph shows multiple time series of the number of drivers, front seat passengers and rear seat passengers killed between January 1969 and December 1984. Every line is colour coded for each variable. The feature label is displayed at the end of the time series line.

The second graph is a single time series plot of the number of drivers killed. I created a tooptip feature which displays a tooltip showing the month and the average value of deaths for that particular month when the mouse is hovered over that point. I also plotted dashed lines mapping to the coressponding values on the x and y axis.

The third graph shows a stacked plot for the number of drivers killed. It also shows a smaller version of the plot at the bottom. I added brushing functionality that allows the user to brush through a region on the small graph and this region is then focussed on the main graph above it. In order to make the graph further interactive, I implemented a panning technique which gives the user flexibilty to move around the brushed region (also moving the corresponding region in the main graph). This context and focus functionality allows the user to get a deeper understanding of the data.
