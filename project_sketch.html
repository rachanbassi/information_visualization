Project Sketch

Planned Tools:

All the visualizations for this project will be implemented in D3 exclusively. The student performance's dataset is quite clean - no missing values, duplicates, etc. I will relabel the column labels in R in order to give them a more descriptive name. So far, I don't think there will be any other elaborate preprocessing required, however, if there are any necessary changes that need to be made to the data, I will execute those changes in either R or Python (depending upon the preprocessing required).

Planned Techniques:

Most of the variables in this dataset are categorical, thus I have chosen relevant visualization techniques for categorical variables.
  • Stacked Bar Chart: I will start by plotting a stacked bar chart of the grades versus absences, with the stacks representing the number of study hours put in (categorical varible divided into four categories based on the number of hours put in. This will give a good understanding of how absences, grades and study hours form a relationship.
  • Parallel Coordinates: Since the dataset is relatively high-dimensional, plotting every row in the data into a parallel cooridinates setup will allow a better understanding of any visible patterns and how the variables 'map' to on each other. In particular, I would like to see how the number of absences, being in a romantic relationship, number hours spent as free, number of hours spent studying, number of past failures, map to the final grade a student receives.
  • Small Multiples: Here I will implement a small multiples representation of the dataset in terms of bar charts. Every barchart will be plotted for a particular categorical variable and will be split into male and female for each category in this variable. Variables of interest here will be the same, however this graph will represent the distributions of these variables over male and female students.
  • Parallel Sets: Time permitting, in addition to the charts mentioned above, I will also try to implement a parallel sets representation of this data - similar to the example here.

Planned Interaction:

In terms of interactivity, for the stacked bar chart, I will provide the user with a tooltip feature will displays the grade when the mouse is hovered over a stack in the chart. I will also highlight the stacked region. For the parallel coordinates, I will implements brushing, allowing the user to select a region on a particular axis and highlight it across all other axes. Finally, for small multiples, I wish to include a tooltip displaying the exact value of the variable for that bar. I will also colour male and female bars with different colour so that, for example, when the mouse if hovered over a bar for male, the corresponding male bars in the other multiples remain active while the female bars fade to grey.
